
# 🌿 Meet A Flora

A Flutter app that identifies plants using AI and shows dynamic plant details.

---

## 📱 Features

- 📸 Take photo using camera
- 🖼️ Select images from gallery
- 🌿 Plant identification using PlantNet API
- 🤖 AI-generated plant details using Gemini API
- 🔄 Dynamic fresh data every time
- ⚡ Clean UI with BLoC state management

---

## 🏗️ Architecture

This project uses Clean Architecture:

- Presentation (BLoC)
- Domain (Use Cases + Entities)
- Data (Repositories + Data Sources)
